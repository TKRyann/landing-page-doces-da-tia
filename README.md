# 🍰 Projeto Landing Page — Doces da Tia

Este é um projeto de uma **Landing Page institucional** desenvolvido para a confeitaria artesanal *Doces da Tia*. O objetivo principal foi criar uma interface limpa, atraente e totalmente responsiva para apresentar o trabalho da marca e facilitar o contato dos clientes.

O projeto foi construído do zero utilizando apenas **HTML5 e CSS3 puros**, focado em aplicar boas práticas de estruturação e estilização sem o uso de frameworks.

---

## 📦 Funcionalidades da Interface

O design reproduz fielmente uma experiência profissional de e-commerce e portfólio gastronômico:

* **Menu Superior Institucional:** Links de navegação dinâmica (`Início`, `Cardápio`, `Sobre mim`) e botão de destaque para conversão imediata (`Peça aqui`).
* **Seção Hero Impactante:** Frase de efeito com tipografia bem trabalhada, parágrafo institucional envolvente e dois botões de ação principais (`Ver cardápio` e `Telefone integrado`).
* **Barra de Redes Sociais:** Botões minimalistas com ícones de redirecionamento rápido para canais oficiais: **WhatsApp**, **Instagram** e **Facebook**.
* **Design de Selo Identitário:** Elemento circular com a logo oficial e o slogan *"Seu dia mais doce"*.

---

## 🎯 Objetivos do Projeto

* **Prática de Clean Code:** Desenvolver uma estrutura HTML semântica e organizada.
* **Fidelidade de Design:** Reproduzir fielmente a identidade visual da marca através do CSS (cores, tipografia e espaçamentos).
* **Foco em Conversão (UI/UX):** Posicionamento estratégico de botões de chamada para ação (CTAs) e links para redes sociais (WhatsApp, Instagram e Facebook).

---

## 💻 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica da página (`<header>`, `<nav>`, `<section>`).
* **CSS3:** Estilização, layout (Flexbox/Grid), design responsivo e efeitos de transição nos botões.
